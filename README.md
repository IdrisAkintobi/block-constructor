# Bitcoin block constructor

## Description

Btrust block constructor challenge solution.

### Requirements

-   [Node.js](https://nodejs.org/en/) v20.0.0 or higher
-   [NPM](https://www.npmjs.com/) v9.0.0 or higher

#### Installation

```bash
$ npm install
```

#### Running the app in development mode

```bash
$ npm run dev
```

#### Build the app

```bash
$ npm run build
```

#### Running the app

```bash
$ npm run start
```

#### Test

```bash
$ npm test
```
